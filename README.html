Windows CAN Utils

Provides CLI tools for interacting with Cyder CAN modules over serial. We currently target the DSD TECH SH-C30A USB to CAN adapter on Windows devices.

Installation

Download & run the latest .msi installer from Releases

Usage

CAN Server

Opens a serial line CAN connection to an slcan adapter and exposes it via Windows pipe.
To find the adapter, open 'Device Manager' and look in the 'Ports (COM & LPT)' dropdown.

Usage: can_send <port> [--bitrate <ID#DATA>]
can_server COM5 --bitrate 1000000

CAN Dump

Prints realtime CAN data from an open CAN pipe to standard output.

can_dump COM5

NOTE: A CAN server instance must be open for the target device.

CAN Send

Sends the given CAN frame to an open CAN pipe.

Usage: can_send <port> <ID#DATA>
can_send COM5 055#00

NOTE: A CAN server instance must be open for the target device.

Developers

Install from source files (Using Cargo)

cd win_can_utils
cargo install --path .

Generating MSI Installer

To install cargo wix:

cargo install cargo-wix

Download the WiX3 binaries .zip - https://github.com/wixtoolset/wix3/releases
Extract it somewhere on your PC. ie: C:\Program Files\WiX Toolset v4.0\bin
Add the directory containing the binaries to PATH in Environment Variables.

Generate MSI file:

cargo wix
